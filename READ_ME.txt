plagiargiarismCatcher
This program takes a directory path of files and computes their similarity.

How to use:
unzip cheaters_kah4285.zip (This will restore the files main.cpp and Table.h)
make (This will compile the files, create main.o, and create the executable program plagiarismCatcher)

./plagiarismCatcher path/to/files #(WordsperChunks) #(SimilaritiesForComparison)

File1
File2 - Plagiarized
File3 - Not Plagiarized

Output:
High#: File1, File2
